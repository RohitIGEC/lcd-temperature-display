# LCD Temperature Display

## How it Works
- TMP36 sensor reads temperature
- LCD 16x2 displays real time temperature
- Temperature normal → "Normal" message
- Temperature > 30°C → "WARNING: HOT!" message

## Components
- Arduino Uno
- TMP36 Temperature Sensor
- LCD 16x2 Display

## Real World Use
- Digital thermometers
- AC temperature displays
- Industrial temperature monitors
- Smart home systems

## Skills Used
- Embedded C programming
- LCD interfacing
- Analog sensor reading
- Real time display update
